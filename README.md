# Simple Console Chat

One session only. No history.

### Installation

```bash
git clone https://github.com/jmpleon/simple-console-chat.git
cd ./simple-console-chat
```

### Build

```bash
g++ .\main.cpp .\chatman.cpp .\dataman.cpp -o simple-console-chat
```

### Run

```bash
./simple-console-chat
```
